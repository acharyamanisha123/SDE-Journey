# AI Resume Mentor & Analyzer 🚀
A smart web tool built to help developers optimize their resumes using data-driven insights.

### Key Features:
- **Keyword Analysis:** Scans for Power Verbs and Tech Skills.
- **Frequency Mapping:** Identifies the "Top Skill" using an O(1) Hash Map.
- **Real-time UX:** Live word counting and dynamic scoring progress bars.
- **Gap Analysis:** Suggests missing keywords to improve ATS compatibility.

### Tech Stack:
- HTML5, CSS3, JavaScript (ES6+)
- GitHub Pages for Deployment
